═══════════════════════════════════════════════════════════════════════
  WHAT IS POWERPOSE?
═══════════════════════════════════════════════════════════════════════

PowerPose is a panel-based posing tool inspired by DAZ Studio's PowerPose.
Instead of clicking in the 3D viewport, you click control point BUTTONS in
a panel and drag your mouse to rotate bones directly.

  KEY FEATURES:
  - 2D panel interface (N-panel sidebar)
  - Dual mouse button control (left=bend, right=twist)
  - Real-time viewport updates
  - Automatic keyframing
  - Works alongside DAZ Bone Select (not a replacement!)

═══════════════════════════════════════════════════════════════════════
  BASIC USAGE
═══════════════════════════════════════════════════════════════════════

  LEFT-CLICK = BEND (Perpendicular Rotation)
  ┌─────────────────────────────────────────┐
  │  1. Left-click "Left Forearm" button    │
  │  2. Drag mouse up/down or left/right    │
  │  3. Watch elbow bend in viewport        │
  │  4. Release mouse to keyframe           │
  └─────────────────────────────────────────┘

  RIGHT-CLICK = TWIST (Length Rotation)
  ┌─────────────────────────────────────────┐
  │  1. Right-click "Left Forearm" button   │
  │  2. Drag mouse horizontally             │
  │  3. Watch forearm twist                 │
  │  4. Release mouse to keyframe           │
  └─────────────────────────────────────────┘

  CANCEL = ESC KEY
  ┌─────────────────────────────────────────┐
  │  1. Click and drag control point        │
  │  2. Changed your mind?                  │
  │  3. Press [ESC] key                     │
  │  4. Rotation reverts to original        │
  └─────────────────────────────────────────┘

  UNDO = CTRL+Z
  ┌─────────────────────────────────────────┐
  │  1. Keyframed a rotation                │
  │  2. Don't like it?                      │
  │  3. Press [Ctrl+Z]                      │
  │  4. Standard Blender undo works!        │
  └─────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════
  MOUSE DRAG BEHAVIOR
═══════════════════════════════════════════════════════════════════════

  HORIZONTAL DRAG:    Rotates around vertical or perpendicular axis
  │                   Example: Head turns left/right
  │  ───────>
  │
  └─────────────

  VERTICAL DRAG:      Rotates around horizontal or secondary axis
  │         ▲         Example: Head nods forward/back
  │         │
  │         │
  └─────────────

  DIAGONAL DRAG:      Combines both rotations
  │        ╱          Example: Complex head tilt
  │      ╱
  │    ╱
  └─────────────

  SENSITIVITY: 0.01 radians per pixel (~0.57 degrees per pixel)
  - Drag slowly = Fine control
  - Drag quickly = Large rotations

═══════════════════════════════════════════════════════════════════════
  TIPS & TRICKS
═══════════════════════════════════════════════════════════════════════

  ✓ START SLOW: Drag slowly at first to get a feel for sensitivity

  ✓ USE BOTH BUTTONS: Left-click for bends, right-click for twists

  ✓ ESC IS YOUR FRIEND: Don't like the rotation? Just press ESC!

  ✓ COMBINE WITH IK: Use DAZ Bone Select for rough pose, PowerPose
    for fine-tuning

  ✓ ONE BONE AT A TIME: PowerPose rotates only the clicked bone,
    unlike IK which affects entire chains

  ✓ REAL-TIME FEEDBACK: Watch the viewport as you drag - what you
    see is what you'll keyframe


═══════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

  PROBLEM: Panel doesn't appear
  SOLUTION: Press [N] key, click DAZ tab, ensure addon is enabled

  PROBLEM: Control points don't work
  SOLUTION: Ensure you're in Pose Mode with armature selected

  PROBLEM: Wrong bone rotates
  SOLUTION: Check bone names match Genesis 8 (lHand, rFoot, etc.)

  PROBLEM: Rotations too sensitive
  SOLUTION: Drag more slowly for fine control (sensitivity will be
            adjustable in future phase)

  PROBLEM: Rotation feels wrong
  SOLUTION: Try the other mouse button - left for bend, right for twist


═══════════════════════════════════════════════════════════════════════
  KEYBOARD SHORTCUTS
═══════════════════════════════════════════════════════════════════════

  [N]           - Open/close N-panel sidebar
  [ESC]         - Cancel rotation (during drag)
  [Ctrl+Z]      - Undo last rotation
  [Ctrl+Shift+Z]- Redo rotation

  Note: PowerPose has NO dedicated hotkey - just open N-panel > DAZ tab
