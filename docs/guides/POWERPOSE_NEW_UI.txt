═══════════════════════════════════════════════════════════════════════
  WHY TWO BUTTONS?
═══════════════════════════════════════════════════════════════════════

  PROBLEM: Right-click opens context menu in Blender
  SOLUTION: Use explicit buttons instead

  BENEFITS:
  ✓ Clear and obvious what each button does
  ✓ No conflicts with Blender's context menu
  ✓ Works with any input device
  ✓ More discoverable (no hidden functionality)
  ✓ Standard Blender UI pattern
  ✓ Accessible to new users
