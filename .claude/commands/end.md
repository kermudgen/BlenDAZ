End-of-session ritual for BlenDAZ. Do the following:

1. Review the session — scan the conversation to understand what was worked on and what changed.

2. Identify the active sub-project — was this session mainly PoseBridge, PoseBlend, or root BlenDAZ work? (May be multiple.)

3. Update the active sub-project SESSION_START.md:
   - PoseBridge: `posebridge/SESSION_START.md`
   - PoseBlend: `poseblend/SESSION_START.md`
   - Update: **Updated** date, **Current State** (2-3 sentences), **What We Did Last Session** (replace not append), **Next Up** (sync from TODO.md), **Don't Forget** (add new gotchas, prune stale), **Files Most Likely Needed Today**

4. Update root SESSION_START.md (`SESSION_START.md`):
   - Update: **Updated** date, **Current State**, **What We Did Last Session**
   - Update the **Active Sub-Projects** status table if phase or status changed
   - Update **Next Up**

5. Rules:
   - Replace sections, never append — "What We Did Last Session" is THIS session only
   - Keep each file under ~100 lines
   - History goes in SCRATCHPAD.md, not SESSION_START.md

6. When done, output only: "✓ SESSION_START.md updated. Run /clear to start fresh."
